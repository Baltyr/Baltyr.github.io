Soft Sleeping

Este proyecto fue uno de los primeros desafíos que enfrenté en mi carrera. Se trató de una página web enfocada en la regulación del sueño y el mejoramiento de tu día a día gracias a esto. Fue mi primera experiencia en el desarrollo de páginas web, utilizando únicamente las tecnologías básicas de HTML y CSS. Además, realicé la creación de mockups de la aplicación para asegurar una buena experiencia de usuario.

Synergym

Este proyecto consistió en la creación de una página web para un gimnasio, la cual ofrece una variedad de características y servicios para sus usuarios. Además, se incluyó un sistema de login para que los usuarios registrados pudieran acceder a un menú especial con opciones personalizadas. El proyecto fue desarrollado como trabajo de defensa de título y se utilizó una combinación de tecnologías web como HTML, CSS, JavaScript y PHP para su implementación.
